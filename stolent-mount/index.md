Room Link: Stolen Mount

Challenge Explanation
In this scenario, an intruder infiltrated a corporate network and targeted an internal Network File System (NFS) server where backup files were stored. A classified secret was accessed and exfiltrated, leaving only a packet capture (PCAP) file recorded during the incident. The objective of this challenge is to perform network traffic analysis on the provided PCAP, identify the exfiltrated data, reconstruct the stolen files from the raw packet bytes, and ultimately recover the classified flag.
